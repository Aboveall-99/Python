# Python

A collection of small Python practice scripts covering algorithms, basic data
science, and OOP fundamentals.

## Contents

| File | What it does |
|---|---|
| `Factorial.py` | Recursive factorial with execution-time measurement across input sizes |
| `Prime generator.py` | Prints primes up to a user-given upper limit |
| `Two sum.py` | Two Sum problem (brute-force attempt, work in progress) |
| `Two sum using pointers.py` | Two Sum via a two-pointer approach (in progress) |
| `Two sum using hashmaps.py` | Two Sum via a hashmap approach (in progress) |
| `Dictionary.py` | Builds a small ad-spend dataset with pandas and exports it to `cleaned_data.csv` |
| `Regression_line.py` | Fits a scikit-learn `LinearRegression` model on the ad-spend data |
| `testclasses.py` | Basic class/OOP example (`Calc` class with add/sub/mult/div) |
| `functions.py` / `example.py` | Function basics and code-reuse example |
| `new_project.py` | Loop and list practice |

## Status

`Two sum.py`, `Two sum using pointers.py`, and `Two sum using hashmaps.py` are
unfinished/in progress.

## Requirements

`Dictionary.py` and `Regression_line.py` need `pandas`, `matplotlib`, and
`scikit-learn`:

```
pip install pandas matplotlib scikit-learn
```
